# Trabalho sobre Git Hub
## Git Hub
São ferramentas que por meio deles eu consigo mandar meu projeto do VsCode para o Git Hub para obter mais praticidade.

# Passo a passo

* Criar uma pasta 
* abra ela com VsCode 
* Crie um README.md 
* Abra o terminal
* De uma "versão git" para verão 
* De um "git init"
* De um git config --global user.name "Bruno" 
* De um git config --global user.email "engs-brunocaldas@camporeal.edu.br"
* De um git status git 
* De um git add .
* De um git commit -m ""
